# Daily Sales Checklist (Solo Founder)

## 1) Prospecting (30–45 min)
- [ ] Add 10 qualified prospects
- [ ] Capture contact + channel (email/X/Reddit)
- [ ] Add 1 personalization line per prospect

## 2) Outreach (45–60 min)
- [ ] Send 20 new messages
- [ ] Use at least 2 template variants
- [ ] Log sends in KPI tracker

## 3) Follow-Up (20–30 min)
- [ ] Follow up with leads from 2–3 days ago
- [ ] Answer all inbound replies same day
- [ ] Push qualified leads to “send one-pager”

## 4) Conversion (20 min)
- [ ] Send pilot one-pager to warm leads
- [ ] Ask direct close question: “Start pilot this week?”
- [ ] Confirm assets + billing email for yes responses

## 5) Metrics + Improvement (15 min)
- [ ] Update KPI CSV
- [ ] Note best-performing channel/template today
- [ ] Pick 1 change to test tomorrow
